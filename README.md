# EuropeanBankChurnAnalysisProject-unifiedmentor
European Bank Customer Churn Analytics Project | EDA, Segmentation, KPI Framework &amp; Interactive Streamlit Dashboard.
## Objective

The objective of this project is to analyze customer churn behavior in a European bank using exploratory data analysis (EDA), KPI modeling, and interactive dashboarding.

The project aims to identify high-risk customer segments, understand churn drivers, and provide data-driven business recommendations to improve customer retention.
## Tech Stack

- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Streamlit
- Jupyter Notebook
- Git & GitHub
## Data Source

This project was assigned as part of my Data Analyst Fellowship at Unified Mentor.

The dataset was provided during the internship for analytical and academic purposes. 
It contains structured customer-level banking data used to perform churn analysis and KPI modeling.
## Features & Highlights

- Performed Exploratory Data Analysis (EDA)
- Designed churn-related KPIs
- Created Credit Score Risk Bands
- Analyzed Active vs Inactive Member churn
- Built Engagement Gap KPI
- Conducted Geography-wise churn comparison
- Developed an interactive Streamlit dashboard
- Documented findings in a structured research paper format
